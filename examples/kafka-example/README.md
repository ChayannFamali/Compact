# compact — Kafka пример

Показывает типобезопасную отправку и получение сообщений через Kafka.

## Что демонстрирует

- Контракт определён один раз, используется продюсером и консьюмером
- Продюсер автоматически добавляет `X-Compact-Contract-*` заголовки
- Консьюмер проверяет версию контракта перед десериализацией
- Изменение case class без изменения контракта → ошибка компиляции

## Быстрый старт (3 команды)

```bash
# 1. Запустить Kafka
docker-compose up -d

# 2. Запустить консьюмера (в первом терминале)
sbt "kafkaExample/runMain io.compact.example.kafka.ConsumerApp"

# 3. Запустить продюсера (в другом терминале)
sbt "kafkaExample/runMain io.compact.example.kafka.ProducerApp"
